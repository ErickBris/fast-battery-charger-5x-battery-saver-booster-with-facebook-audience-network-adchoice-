A) Fast Charger Activity - top

1.Background : res > drawable > background.xml

        android:angle="135"
        android:centerColor="#11927e"
        android:endColor="#0a5193"
        android:startColor="#109060"

2.Card Color : res > values > color.xml

name="cardcolor"

Replace color value with #AARRGGBB or #RRGGBB

3.Battery-Color : src > main > java > fast.battery.charger.app > FastCharger.java

Line 296 : waveProgressView.setText("#4D000000", 70);
Line 297 : waveProgressView.setWaveColor("#4D76FF03");

Replace color and size for change battery level text : waveProgressView.setText("YOUR COLOR", TEXT SIZE);

Replace wave color for change battery wave : waveProgressView.setWaveColor("YOUR COLOR");

4.Icon : res > drawable > ic_tool_*.png

Replace current tools icon with your new icon

5.Button Start fast charger : res > drawable > ic_button.9.png & res > drawable > ic_button_press.9.png

Replace current image icon with your new image

Please prefer this link for know about 9 patch image : https://developer.android.com/studio/write/draw9patch.html

Please prefer this tool to create 9 patch image : https://romannurik.github.io/AndroidAssetStudio/nine-patches.html

B) Main Activity - top

1.Circle Progress View : res > layout-large,layout-normal,layout-xlarge > activity_main.xml

app:arc_bottom_text_color="@color/White"
